Instrukcja startu

Kroki do uruchomienia MVP.

  1. Utwórz repo GitHub i wrzuć cały katalog projektu.
  2. Zainstaluj lokalnie zależności: python -m venv .venv, potem pip install -r requirements.txt.
  3. Wygeneruj YouTube refresh token: pobierz OAuth Client JSON z Google Cloud jako client_secret.json, potem uruchom python scripts/create_youtube_token.py.
  4. Dodaj GitHub Secrets: Gemini, Pexels, YouTube i opcjonalnie Telegram.
  5. Test bez uploadu: Actions -> AutoKanał YouTube -> Run workflow -> no_upload=true.
  6. Publikacja: ustaw YOUTUBE_PRIVACY_STATUS=private na start, sprawdź film, potem zmień na public.
